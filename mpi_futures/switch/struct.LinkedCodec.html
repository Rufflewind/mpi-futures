Obtain a Stream of future incoming messages from the given source. Each message is decoded using the given codec.

fn incoming(&self, Source) -> Stream<Future<Codec::Message>>;

The stream will keep running until the Switch is closed, but you can stop the Stream at any time if you aren't expecting to receive messages. You can even create a new incoming stream every time you want to receive a message.

Just try to avoid running multiple overlapping incoming streams simultaneously, as that could cause messages to be split between the streams in a non-deterministic manner.
